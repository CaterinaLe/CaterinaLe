用matlab画出同一轨道平面下的三体运行轨迹

matlab 三体
技
更新日期:   2020-02-18
文章字数:   1.1k
阅读时长:   5 分
阅读次数:  

一直想模拟一下三体运行，这次终于用matlab做了个小程序，代码附在文后。

思路

假设在极短的时间dt内，天体的速度(vx,vy)不会改变，则dt时间后天体的坐标x’=x+vxdt，y’=y+vydt；
假设在极短的时间dt内，天体的加速度(ax,ay)不会改变，则dt时间后天体的速度vx’=vx+axdt，vy’=vy+aydt；
天体受到的引力由万有引力公式求得，除以质量就是加速度，这个都会的，不说了。

代码

主程序：

clc 
clear
dt=10;%极短时间Δt，在此时间内，速度、加速度认为不变，越小越精确
n=1000;%每经过n*dt时间在图上画出三个天体的位置，合理增大作图间隔时间能节约时间，推荐n*dt=10000
sunm=1.898*10^30;%质量参考值1：太阳质量
earm=5.965*10^24;%质量参考值2：地球质量
moonm=7.36*10^22;%质量参考值3：月球质量
L=1.5*10^11;%坐标参考值1：日地距离
r=3.84*10^8;%坐标参考值2：月地距离
v=92445;%速度参考值1
v2=3239;%速度参考值2
ob1=my_planet(1*sunm,0,0,0,0,0,0);%my_planet(质量，x坐标，y坐标，x轴方向的速度，y轴方向的速度，x轴方向的加速度，y轴方向的加速度)
ob2=my_planet(1*sunm,L,0,0,v,0,0);
ob3=my_planet(1*sunm,L,L,-v,v,0,0);
i=0;
while 1
    [ax21,ay21]=gra(ob2,ob1);%天体2对天体1贡献的加速度
    [ax31,ay31]=gra(ob3,ob1);%天体3对天体1贡献的加速度
    [ax12,ay12]=gra(ob1,ob2);%天体1对天体2贡献的加速度
    [ax32,ay32]=gra(ob3,ob2);%天体3对天体2贡献的加速度
    [ax13,ay13]=gra(ob1,ob3);%天体1对天体3贡献的加速度
    [ax23,ay23]=gra(ob2,ob3);%天体2对天体3贡献的加速度
    ob1.ax=ax21+ax31;%天体1沿x轴的加速度
    ob1.ay=ay21+ay31;%天体1沿y轴的加速度
    ob2.ax=ax12+ax32;%天体2沿x轴的加速度
    ob2.ay=ay12+ay32;%天体2沿y轴的加速度
    ob3.ax=ax23+ax13;%天体3沿x轴的加速度
    ob3.ay=ay23+ay13;%天体3沿y轴的加速度
    ob1.x=ob1.x+ob1.vx*dt;%x=x+vx*dt
    ob1.y=ob1.y+ob1.vy*dt;%y=y+vy*dt
    ob2.x=ob2.x+ob2.vx*dt;
    ob2.y=ob2.y+ob2.vy*dt;
    ob3.x=ob3.x+ob3.vx*dt;
    ob3.y=ob3.y+ob3.vy*dt;
    ob1.vx=ob1.vx+ob1.ax*dt;%vx=vx+ax*dt
    ob1.vy=ob1.vy+ob1.ay*dt;%vy=vy+ay*dt
    ob2.vx=ob2.vx+ob2.ax*dt;
    ob2.vy=ob2.vy+ob2.ay*dt;
    ob3.vx=ob3.vx+ob3.ax*dt;
    ob3.vy=ob3.vy+ob3.ay*dt;
    if mod(i,n)==0 %每经过n*dt时间在图上画出三个天体的位置，合理增大作图间隔时间能节约时间，推荐n*dt=10000
        plot(ob1.x,ob1.y,'r.'),pause(0.000001);
        plot(ob2.x,ob2.y,'g.'),pause(0.000001);
        plot(ob3.x,ob3.y,'b.'),pause(0.000001);
        hold on;
        grid on;
    end
    i=i+1;
end

计算加速度(新建函数文件，文件名gra.m)：

function [ax,ay] = gra(ob1,ob2)%求天体ob1对ob2贡献的加速度
    G=6.754*10^(-10);%万有引力常量（瞎设的）
    R=sqrt((ob1.x-ob2.x)^2+(ob1.y-ob2.y)^2);%距离
    a=G*ob1.m/(R^2);%加速度大小
    ax=a*(ob1.x-ob2.x)/R;%沿x轴方向的加速度
    ay=a*(ob1.y-ob2.y)/R;%沿y轴方向的加速度
end

定义行星（天体）类（新建类文件，文件名my_planet.m）：

classdef my_planet %行星类
    properties
        m;%质量
        x;%x坐标
        y;%y坐标
        vx;%x轴方向的速度
        vy;%y轴方向的速度
        ax;%x轴方向的加速度
        ay;%y轴方向的加速度
    end

    methods
        function ob= my_planet(m,x,y,vx,vy,ax,ay)
            ob.m=m;
            ob.x=x;
            ob.y=y;
            ob.vx=vx;
            ob.vy=vy;
            ob.ax=ax;
            ob.ay=ay;
        end
    end

end
