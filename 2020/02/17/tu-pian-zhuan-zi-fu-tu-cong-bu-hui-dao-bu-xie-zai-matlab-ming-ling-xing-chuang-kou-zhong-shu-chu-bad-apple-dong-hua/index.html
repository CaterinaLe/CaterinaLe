图片转字符图：从不会到不屑———在matlab命令行窗口中输出bad apple动画

　　某日同学给我看dy上一个短视频，是将视频转换成ASKII字符输出，这都是十年前的网友玩的了，怎么能难倒我呢，我决定用matlab也做一个。不过饭要一口一口吃，我先做了一个小程序，读取电脑中的图片，转换为字符图，输出到matlab命令行窗口并写入txt文件中。

原理

　　每个字符点阵，比如“W”和“-”,它们黑色所占的比例不同，给人视觉上带来的效果就是“W”比“-”更黑，我们用“&#@”等代替图片中较黑（灰度值低）的像素，用“`-.”等代替图片中较白（灰度值高）的像素，这样，图片就被转换成了字符图。那么，合理的灰度值-字符对应表就是整个程序的关键。我采用的灰度-字符变换关系如下所示：

黑白字符图

　　matlab命令行窗口行距比较大，所以txt显示效果较好，不过输入图片不能过大，因为txt每行的字数有上限，超过就会自动换行，导致图像错乱；而matlab命令行倒不会因为字数限制而导致图像错乱，而它的行距过大的问题可以用每三行像素输出一行字符的办法解决，不过这样也牺牲了行像素。这两种方案都不是特别完美，我更喜欢小图用txt显示，大图用命令行显示，最终效果如下所示（字体设为宋体，字号1）：

色幻觉

　　上图的那种程度不算什么,毕竟彩色图最后变成黑白的字符，总感觉少了什么。然后我在知乎上找到了超神奇的灰度-字符转换关系，如下所示：

　　效果：

　　仔细看看，是不是感觉有点奇怪：这张图好像是彩色的！
　　把字号改大到6再看看：

　　没错，黑白字符呈现出了彩色的图像。那个知乎答主说这是一种色幻觉，不过具体原因他没有给出，我也没查到可信的说法，所以此处暂时按下不表。

字符动画

　　搞定了字符图的代码，做一个字符动画简直是小菜一碟，只要读取视频后逐帧处理输出就好了。本着“可以输出的地方就有bad apple”的原则，第一个字符视频就是bad apple了，这也是本站首页的视频：

　　可惜的是这个程序不能播放声音，bgm是我后期加的，而且不同的灰度-字符程序对于黑白二色视频来说，效果都不会太差。接下来看看色彩更丰富的fate hf -狗哥vs王哈桑：

　　……效果确实差强人意，不过这火花确实蛮好看😂。
　　总之，视频转字符并没有看起来那么难，这也是我从一窍不通到快速掌握并感到不屑的原因，明明matlab可以做很多更好玩的事情，但是为什么我的同学没有看到并分享给我！

代码

灰度转字符（第一种变换）：

clc
clear 
close all
p='$@B8&M#*oahkbdpqwmO0QLCJUYXzcunxrjft/\|()1{}[]?-_+~<>i!l;:,\"^`. ';%灰度-字符转换关系
%% 读入图像并显示 %%
data=imread('img02.jpg');%读取图片
figure
imagesc(data)%显示原图片
axis equal
Gray_data=rgb2gray(data);%彩色图像转灰度图
figure
imagesc(Gray_data)%显示灰度图
axis equal
colormap gray
%% 按照灰度矩阵生成字符矩阵 %%
[m,n]=size(Gray_data);
for i=1:m
    for j=1:n
        flag=fix(Gray_data(i,j)/4)+1;
        Gray(i,j)=p(flag);
        if mod(i,3)==1
            Gray3((i+2)/3,j)=p(flag);%在命令行窗口中每三行像素输出一行字符
        end
    end
end
Gray3%在命令行窗口输出
%% 写入文本文件 %%
dlmwrite('test2.txt',Gray,'delimiter',' ','newline','pc')

灰度转字符（第二种变换）：

clc
clear 
close all
p='$&m8p*aou!.';%灰度-字符转换关系
%% 读入图像并显示 %%
data=imread('img02.jpg');%读取图片
figure
imagesc(data)%显示原图片
axis equal
Gray_data=rgb2gray(data);%彩色图像转灰度图
figure
imagesc(Gray_data)
axis equal
colormap gray
%% 按照灰度矩阵生成字符矩阵 %%
[m,n]=size(Gray_data);
for i=1:m
    for j=1:n
        flag=fix(Gray_data(i,j)/25);
        Gray(i,j)=p(flag+1);
        if mod(i,3)==1
            Gray3((i+2)/3,j)=p(flag+1);%在命令行窗口中每三行像素输出一行字符
        end
    end
end
Gray3%在命令行窗口输出
%% 写入文本文件 %%
dlmwrite('test.txt',Gray,'delimiter',' ','newline','pc')

字符动画：

clc
clear
close all
obj = VideoReader('1.mp4');%输入视频位置
obj_numberofframe = obj.NumberOfFrame;%读取总的帧数
obj_height = obj.Height;%读取视频帧高度
obj_width = obj.Width;%读取视频帧宽度
gr='$@B8&WM#*oahkbdpqwmZO0QLCJUYXzcvunxrjft/\|()1{}[]?-_+~<>i!lI;:,\"^`. ';%灰度-字符转换关系
for f=1:1:obj_numberofframe
    frame = read(obj, f);%读取第f帧画面
    Gray_data=rgb2gray(frame);%彩色图像转灰度图
    for ii=1:3:obj_height%在命令行窗口中每三行像素输出一行字符
        for j=1:1:obj_width
            flag=fix(Gray_data(ii,j)/4)+5;
            i=(ii+2)/3;
            Gray(i,j)=gr(flag);%在命令行窗口中每三行像素输出一行字符
        end
    end
    imshow(frame);%显示第f帧画面
    Gray%在命令行窗口输出
end
